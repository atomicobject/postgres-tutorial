# Postgres Workshop

## Prerequisites

- Docker Desktop (running)
- Node.js 20+
- npm

## Install

```bash
cd frontend
npm install
```

## Run

Make sure Docker Desktop is running, then:

```bash
docker compose up --build
```

## Access

- **App**: http://localhost:3000
- **pgweb** (database UI): http://localhost:8081
